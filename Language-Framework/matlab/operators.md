# matlab 运算符

| 操作符  | 意义                      |
| ---- | ----------------------- |
| +    | 相同size的array相加，或者某一个是标量 |
|      | 减                       |
|      | 矩阵乘法，线性代数中的矩阵乘法         |
| .*   | 数组乘法，对应元素相乘，或者某一个为标量    |
| /    | 矩阵除，A/B等价于A*inv(B)      |
| ./   | 数组除法，对应元素相除，或者某一个是标量    |
| \    | 矩阵左除，A\B等价于inv(A)*B     |
| .\   | 数组左除。                   |
| ^    | 矩阵的幂                    |
| .^   | 数组的幂，对应位置的幂             |
| '    | 矩阵转置                    |
| .'   | 数组转置                    |
