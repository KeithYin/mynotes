# skimage 库简单介绍

**安装：** `pip install scikit-image`

`from skimage import data, io, filters, transform`

* data : 图片数据模块，存放一些图片数据
* io  : IO模块，图像的读取，保存，显示
* filters : 各种滤波
* transform : 各种变换都在这
