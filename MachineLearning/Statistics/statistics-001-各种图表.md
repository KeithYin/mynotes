# Q-Q plot

https://zhuanlan.zhihu.com/p/232271603

> Quantile-quantile plot

作用：判断数据是否符合一个分布

如何做图

* 给定数据。如果我们判断该数据是否符合正态分布，那就从正态分布中采样等量的样本点。原始数据 与 从正态分布采样得到的样本点 都从小到大排序。两组数据构成 pair，画散点图，就构成了 Q-Q plot。如果原始数据符合正态分布，那么 Q-Q plot 就可以看到一条直线。
