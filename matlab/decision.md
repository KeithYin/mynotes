# 条件分支

## if else

```matlab
if condition
	statement
end

%--------------

if condition
	statement
else
	statement
end

%--------------
if condition
	statement
elseif
	statement
end

```



# 循环

```matlab
% for loop, values可以是以下形式 initval:endval, initval:step:endval, valArray
for index = values
	statement
end

% while loop
while condition
	statement
end

% 可以loop套loop
% 也有continue和break语句
```
